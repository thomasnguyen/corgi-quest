# Corgi Quest Marketing Content

This folder contains auto-generated marketing content maintained by Kiro hooks.

## Files

### `tweet-ideas.md`
Auto-generated tweet ideas when features are completed.

Generated by: Marketing Content Generator Hook  
Trigger: When tasks are marked complete in `.kiro/specs/**/tasks.md`

### `demo-script.md`
Presentation script for Kiroween demo.

Generated by: Demo Script Generator Hook (manual trigger)  
Trigger: Run manually when preparing for demo

### `changelog.md`
User-facing changelog of completed features.

Generated by: Changelog Generator Hook  
Trigger: When tasks are marked complete

## How to Use

### Generate Tweet Ideas
1. Complete tasks in any spec
2. Mark them [x] in tasks.md
3. Save the file
4. Check `tweet-ideas.md` for new content

### Generate Demo Script
1. Open Command Palette
2. Run "Kiro: Trigger Hook Manually"
3. Select "Demo Script Generator"
4. Review `demo-script.md`

### Update Changelog
Happens automatically when you complete tasks.

## Tips

- Review generated content before posting
